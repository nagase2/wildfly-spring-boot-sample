　データ見える化/解析Demoページ

　 データの解析、分析ソリューションのポータルページです。

# タイトル 説明 備考
Bigデータ解析サンプル No SQLのElasticsearchとKibanaを利用し、９千万件のデータを解析するサンプルです。 ９千万件のデータのデータの解析が数秒で完了することを確認できます。
ダッシュボード ダッシュボードサンプルです。 実現にはhightchartを利用しており、別途ライセンスの購入が必要となる。
多重ラインチャート 多重ラインチャートのサンプルです。複数のデータの解析結果を１つのグラフにまとめることにより、標準から外れたデータの発見ができます。
